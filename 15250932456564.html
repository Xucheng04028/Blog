Windows 下验证哈希值

2017/8/18 comments

Windows 自带的文件校验和完整性验证程序（FCIV）可用于计算一个文件的 MD5 或 sha-1 哈希值。在命令行键入命令 FCIV-md5-sha1 path\filename.ext 以验证，比如 FCIV-md5-sha1 c:\windows\system32\shdocvw.dll。

iamCristYe：如果提示错误，建议使用 certutil -hashfile eg.ext SHA1 或 Get-FileHash eg.ext -Algorithm SHA1（后者需 PowerShell）安装相应工具。
