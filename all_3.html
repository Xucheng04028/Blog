关于订阅制的思考

总的来说，我认为订阅制更多地是件好事：

  1. 抑制盗版市场，遏制冲动消费；
  2. 开发者的收入长期稳定，可以更好地满足使用者的个性化需求。

这对真正需要的客户是件好事，对开发者也是好事。但是，为啥订阅价格那么贵？不论是 1Password 还是 office 365 等，其一年的订阅价格差不多等于买断价格的一半或更多，有的甚至更贵。但是就我而言，我之前购买了 Parallel Desktop 11，至今已使用快有三年。所以，目前我对订阅价格不太满意，但承认订阅制也是比较好的策略。

Ulysses 之前作为最具口碑的 Markdown 写作工具，但它取消了买断版，不像 1Password 仍有独立版在售，这对老用户是较大的伤害。虽然 Ulysses 的开发者写了一篇长文章介绍了为何这么做及订阅制的优势，但我还是觉得他们太武断了。

我还认为，由于鲜有人愿意为很多应用按月或年付账，订阅制会迫使我们不得不依赖越来越少的工具。可能原来我们买了这个买了那个，可以这个用一点那个用一点，但随着订阅制成为主流，未来我们可能主要只使用几个应用。

然而我觉得这也许也是一件好事，因为我们之前可能多数应用都是浅尝辄止，但现在会有动力探究这些应用的高级用法，这对于效率到底是提高还是降低我还不好做判断。我只是觉得应用少了可我们对每个工具都越来越熟悉了，切换工具的成本就可以避免了，我们会更专注于内容创作。

Spotlight 搜索结果禁止显示开发者文件

cd /Applications
touch Xcode.app

然后转到 Spotlight 首选项并取消选中 Developer（可能需要重新启动系统偏好设置）

即使更改设置，您也需要保留虚拟 Xcode.app，删除将导致设置恢复。

另一个解决方案是安装 Xcode 本身。

本文参照：

  1. https://forums.macrumors.com/threads/spotlight-developer-and-web-searches-cant-be-disabled.1193533/
  2. https://gxnotes.com/article/127849.html

touch 命令：一是用于把已存在文件的时间标签更新为系统当前的时间（默认方式），它们的数据将原封不动地保留下来；二是用来创建新的空文件。

来自: http://man.linuxde.net/touch

专业的高颜值 GIF 录制应用：Gifox

本文被少数派精选并推荐至首页：点此查看

无论是开源的 Kap 还是 Mac App Store 中的 GIF Brewery、GIPHY Capture 等，这些 GIF 录制应用我都用过，其共同点是免费，这也是我很久之前选择应用的倾向。后不满于这类应用的体验，在 Mac App Store 和搜索引擎以 GIF 为关键字检索：

  1. GIPHY 等免费 GIF 录制应用较为醒目；
  2. 有一些优秀的应用是录制视频，后将视频转为 GIF，或是直接将图片、视频转为 GIF；
  3. 借助多个应用，如 QuickTime、ffmpeg、gifsicle 三者，也是将视频转为 GIF。

GIPHY 等免费且知名的应用的缺陷是没有窗口录制模式；视频、图片转 GIF 类应用又增加了步骤，而且通常这类应用也不是专为 GIF 设计，故难免臃肿；借助多个应用无疑更增加了工作量。

我同时也发现了 Gifox，当时在 Mac App Store 没有一则评论，检索也很难发现有人推荐。但，大概它是目前唯一一款专为 GIF 录制所设计并支持窗口录屏的应用，专注、轻量、优美、快速。

GIF 窗口录制模式

这也是我给予它最强 GIF 录制工具评价的主要原因之一，GIPHY Capture、Kap、GIF Brewery 等 GIF 录制应用都没有这个功能，它们都只有区域录制功能，这也许是 GIF 界的独苗。而窗口录制模式的体验实在美好。当然，确实也可以选择支持窗口录屏的应用，然后输出为 GIF，它的劣势我前面也讲过。

GIF 录制控制

  1. 设置输出目录，一般桌面是最方便的选择；
  2. 设置图像捕捉方式：勾选 Cursor 表示录制光标，勾选 Windows shadow 表示也录制类似截屏的阴影，勾选 Desktop backgroud 表示录制桌面背景；
  3. 控制压缩比例：亲测设为 0.5 时体积减小了三分之二；
  4. 控制「录制和播放」的每秒显示帧数（FPS），低帧数可以减少 GIF 体积，但也会影响播放流畅度；
  5. 设置播放循环次数及播放间隔。

压缩功能

Gifox 的默认压缩算法总是选择近期最好的两种，老式算法只作为备选方案，我们可以设置具体的选项：

  1. 减小 Max. colours 以缩减 GIF 文件体积，但也会因为色彩过少使照片的观感降低；
  2. 指定 Palettle 即调色板，First frame 指以第一帧为基础生成调色板，好处是可以减小 GIF 体积；
  3. 上一步指定了调色板，需要借助 Dither（抖动）使调色板色彩尽量达到原色彩的程度，GIF 最大只能设置为 256 色调色板，所以需要使用 Dither 以模拟出更多的色彩。形象地讲：比如色彩限定在 100 种，但实际录制的画面包含 1000 种色彩，Dither 可以使 100 种色彩更好地表现出 1000 种色彩的效果，使人难以察觉其中的差别；
  4. 设置 Dither 算法：规则抖动算法 Bayer 抖动因其简单有效而较常用，Bayer scale 是 Bayer 抖动的参数，数值越高 GIF 质量越好。误差分散算法 Floyd-Steinberg 抖动虽然可以产生更好的图像效果，但资源占据较多。若要了解其他算法还请参阅文献。

Gifox 支持一键分享功能，可以绑定自己的 Dropbox、Google Drive 等账号，不像 GIF Brewery 等需要注册它们的账号，而且官网表明新版本会支持 Slack、Box、GIPHY。当然，它也支持设置快捷键以快速录制和停止。如果对 Gifox 仍有疑问，点击界面右下角的 ? 可以看到对功能的详细解释。

但 Gifox 也有不足，无法对录制的 GIF 进行编辑，这只适合一次录制即成品，但也难免出差错，还好可使用预览，来删除无用帧或者添加字幕。关于这点我也会向开发者反馈。

你可以在 Mac App Store 下载 Gifox（30 元），我之前评测过 PDF 应用，曾感慨全五星应用实在难得，但在 GIF 领域 Gifox 就是一款颜值与功能都无可挑剔的应用，如果未来增加编辑功能，那它就无愧于 GIF 领域的全五星应用。


本文应用版本：
Gifox 1.4.0

星际争霸地图凯甘瑞生存教程

对于人类来说，这个是生存游戏，反击是后期的事情。也即，前期无论如何也不是虫族的对手，抱团更是送上门的肉。

人族

科学家
灵魂，我最喜欢的角色，最喜欢气流。
晋升者
游牧民
隐刀（辅助型角色）
先知（辅助型角色）：

  1. 堵路：除水晶之外的所有建筑都是隐形的，所以利用这点可以堵路。一般使用星门堵路。不要在隐形建筑外边造建筑，这样会暴露队友的，虫族 K 可以反隐，且都有雷达探测。
  2. 侦查：视野结界用得好，队友夸你真是好。如前期灵魂塔攻击范围为 5，但视野很小只有 1，所以视野结界是一大助力。在堵路附近隐蔽位置放视野结界也可提前预警，确定合适的跑路时机。
  3. 经济：萃取结界放在队友经济建筑旁边即可，一般最多三次叠加，即一个经济点的萃取结界最好不要超过三个。此外，虫族攻来务必回收结界，因为打碎结界虫族会得到大量瓦斯。
  4. 防守：爆炸结界

2017 年度主力应用

入选原则：

  • 使用频率较高；
  • 系统自带应用还是单列吧；
  • 想到什么就写什么，所以间歇性更新……

目的：

  • 大致看下主力应用的花销，抑制冲动消费；
  • 在不断精减应用之路走得更远。

免费

  1. Wechat & QQ
  2. Chrome
  3. Office：教育版，用于论文的写作
  4. OneNote：个人 Wiki
  5. Ship
  6. Zotero：文献管理工具
  7. Anki
  8. 效率应用：谈不上主力，但确实让我很懒的应用
  9. Desktop Ghost：临时隐藏桌面所有文件图标

收费

  1. Parallel Desktop：Windows 虚拟机，融合模式体验极好
  2. 1Password Pro：macOS 是 Read Only 状态，主要在 iPad Pro 上使用
  3. OmniFocus Pro：尝试用它连接一切
  4. MWeb：最好的 MD 应用
  5. aText：输入当前日期……
  6. PinCap：管理图片
  7. Zoommy：好多无版权限制的图片
  8. 奇点（Singularity）：微博三方客户端，可以永久保存微博，哪怕根据相关政策已删除
  9. 效率应用：谈不上主力，但确实让我很懒的应用
  10. Wondershare PDF Password Remover：移除 PDF 口令
  11. PDFelement：对 PDF 编辑、合并等几乎所有操作
  12. 订阅服务只有 Apple Music（已暂停） & iCloud

系统自带应用

  1. Mail
  2. Notes：存储笔记、日记、隐私照片等
  3. Safari
  4. iTunes：听订阅的音乐

弃用

  1. Day One：无法认同日记应用变为订阅制，也无法接受不再提供 iCloud 同步功能
  2. Saprk：每次关机就它在那里墨迹

Smart Cover 皮革款与普通款的区别

我一直在为 10.5 iPad Pro 物色一款最满意的保护套，在决定之前，我淘宝了一个七八十的保护套，用了一段时间就不满意了。iPad 保护套是必需品，因为它要站起来… 官方保护套是主要选择，但淘宝上也有一些颜值很高的选择。总之，现在我决定先了解下官方保护套皮革款与普通款的区别。

有人说，普通款采用聚氨酯，也即人造革，这是一种优秀的合成材料，有很多优点：质地轻，可抗多种酸碱和有机溶剂的腐蚀，一般而言其制品具有耐磨、耐温、密封、隔音、加工性能好、可降解等优异性能。而且 Apple 不会卖假货，人造革的质量也有保证。

但皮革款的颜值要比普通款高出一个档次，而且作为真皮有较好的透气性。等等，我要透气性干嘛？

当然，以上所谓的优点并不多重要，我最关心的是可清洁性！所以我还要等客服回复，故此文待续。

客服回复说皮革款更易清洁。

Chromium 无法登陆 Google 账号

缺少 Google API 密钥，因此 Chromium 的部分功能将无法使用

安装 Chromium 后却无法登陆成功，也就无法同步书签、扩展等内容，对此表示心急。检索许多却只发现了 Windows 下的解决方案，还好最后发现了这篇文章：El Captain 设置环境变量，虽然标题不显，但内容确实也解决了「Chromium 无法登陆账号」的问题，因无法登陆也是同样的原因，只需添加环境变量即可解决

1）在目录 ~/Library/LaunchAgents/ 创建一个文件 environment.plist，并将以下内容复制进去：

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
  <key>Label</key>
  <string>my.startup</string>
  <key>ProgramArguments</key>
  <array>
    <string>sh</string>
    <string>-c</string>
    <string>
        launchctl setenv GOOGLE_API_KEY AIzaSyCkfPOPZXDKNn8hhgu3JrA62wIgC93d44k
        launchctl setenv GOOGLE_DEFAULT_CLIENT_ID 811574891467.apps.googleusercontent.com
        launchctl setenv GOOGLE_DEFAULT_CLIENT_SECRET kdloedMFGdGla2P1zacGjAQh
    </string>
  </array>
  <key>RunAtLoad</key>
  <true/>
</dict>
</plist>

2）在终端执行 launchctl load ~/Library/LaunchAgents/environment.plist 以使其立即生效。

也可以在 stack overflow 查看原教程。

为何 Office for Mac 字体总变为 M S 明朝

将 Office 更新至最新版本即可。如果检查更新无法检查到 Office 更新资源，比如我的就是，可到官网下载手动安装。目前我更新到 170919 版本，变字体的问题已消除：

  1. 自动变 M S 明朝，如网页复制；
  2. 无法将 M S 明朝字体更改为宋体等字体；
  3. 检查不到 Office 更新资源的问题也消除了。

Office for macOS 确实不如 for Windows，如果需要常用 Office 不建议购买 Mac 系列，或者购买 Mac 系列的同时也购买 Parallel Desktop 来装 Windows 虚拟机。
